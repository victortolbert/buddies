## Vue Build Setup

```bash
# change to the vue directory
$ cd vue

# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

**Static Analysis**

```bash
# CSS and JavaScript
$ npm run lint

# CSS Only
$ npm run lint:style

# Check JS
$ npm run lint:js

# Run Unit Tests
$ npm run test
```
