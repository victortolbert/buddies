## Getting Started

### React

```bash
$ cd react
$ npm install (or pnpm install, yarn, etc)
$ npm run start
```

### Vue

```bash
$ cd vue
$ npm install
$ npm run dev
```

## Presentation

To start the slide show:

```bash
$ cd slides
$ npm install
$ npm run dev
# - visit http://localhost:3030
```

Edit the [slides.md](./slides/slides.md) to see the changes.
